Analysis Logic & Thresholds

PR (Performance Ratio)

  • PR ≥ 85% — Optimal performance
  • 85% > PR ≥ 75% — Performance warning
  • PR < 75% — Significant underperformance
  • * Logic active 30m after production start (Stabilization)

Inverter Temperature

  • Temp ≤ 40°C — Healthy operating range
  • 40°C < Temp ≤ 45°C — Thermal warning
  • Temp > 45°C — Critical overheating

DC Current (Avg of 12 MPPTs)

  • Morning (Start to 12:00): Green ≥ 10A | Yellow ≥ 2A
  • Afternoon (12:00 to 19:00): Green ≥ 5A | Yellow ≥ 0.5A
  • Current < Yellow Thr — Potential string/junction loss

AC Power & Production

  • AC ≥ 0.85 × Plant Avg — Nominal generation
  • 0 < AC < 0.85 × Plant Avg — Underperforming
  • AC = 0 W — Inverter Tripped (Daylight only)
  • * Inverters are evaluated relative to exactly what the rest of the plant produces at that exact moment.
  • * If POA < 50 W/m² and Plant Avg < 1000 W, deviations are ignored.
  • * Downtime threshold: Cumulative off-time ≥ configured minimum
